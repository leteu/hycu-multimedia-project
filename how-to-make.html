사이트 설명

우선 디자인적 구성입니다.
디자인은 간단하게 헤더, 콘텐츠, 푸터로 구성되어 있습니다.

헤더는 콘텐츠에서 스크롤이 발생하더라도 항상 확인하고 클릭할 수 있게 position 스타일을 sticky로 주어 항상 위에 붙어 있을 수 있게 하였습니다.
또한 제가 좋아하는 초록색을 콘셉트로 잡고 시작했습니다.

콘텐츠는 각 메뉴별로 보여주어야 할 콘텐츠를 표출하는 곳입니다.
Home 메뉴의 영상 리스트들은 한눈에 보기 쉽게 카드 형식으로 제작하였습니다.
또한 클릭 시 모달이 뜨며 영상을 볼 수 있게 하였습니다.

푸터는 간단하게 copyright를 추가해보았습니다.

더 이쁘게 해보고 싶었지만 디자인에는 재능이 없어 최대한 열심히 스타일을 넣어 보았습니다.

그리고 코드의 구성입니다.
header 태그에 css style을 넣고 body 태그의 맨 밑에 줄에 js 코드를 작성하였습니다.

js 또한 head 태그에 적어도 되지만 body 태그 맨 밑줄에 적은 이유는
vue를 사용하던 버릇 탓에 template script style 순으로 코드를 작성하는 것이 손에 익었기 때문인 것 같습니다.

window.onload를 사용해서 위에 적는 것이 좋을까 생각했었지만 익숙한 것을 고르기로 했습니다.
